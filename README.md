## Project Description

This project is a simple web page developed using **HTML5** , **CSS3** and **JS** as part of the Web Technology course. The purpose of this task is to understand the basic structure of a web page and implement various CSS properties for designing responsive and visually appealing layouts.

## Features

* Semantic HTML structure
* Navigation bar using Flexbox
* Sticky navigation bar
* Responsive layout
* Hyperlinks for navigation
* Custom colors and spacing
* Organized page sections using `<header>`, `<main>`, and `<div>`

## So Far Technologies Used

* HTML5
* CSS3


## Learning Objectives

Through this task, the following concepts were practiced:

* HTML document structure
* Semantic HTML elements
* CSS selectors
* Flexbox layout
* Margin and padding
* Position properties (`static`, `relative`, `absolute`, `fixed`, and `sticky`)
* Responsive page design
* Styling hyperlinks and navigation menus

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html` or open it using **VS Code Live Server**.
4. Explore the webpage and test the sticky navigation bar by scrolling.

## Outcome

This task demonstrates the implementation of a simple webpage with a functional sticky navigation bar and the application of fundamental HTML and CSS concepts learned in the Web Technology course.
